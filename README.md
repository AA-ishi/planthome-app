# planthome-app
観葉植物のお世話方法を、気軽に検索できる小さなアプリです。

## どんなアプリ？
植物の名前と置いてある場所を選ぶと、  
その植物に合った管理方法（光・水やり・温度など）を表示してくれるアプリです。

ブラウザで開くだけで使えるので、インストールは不要です。  
ブラウザの翻訳機能も使えるので、英語の説明もすぐ日本語にできます。

## できること
- 植物の名前を入力して検索
- 植物の置き場所（明るさなど）を選択
- 次の水やりの目安を確認
- すべてブラウザ上で動作（あなたのPCだけで完結）

※ メモ保存や水やり記録の機能はありません。

## つかいかた
1. アプリのURLをブラウザで開きます  
2. 植物の名前を入力します  
3. 植物の置いてある場所を選びます  
4. 「ここをクリック」ボタンを押すと、管理方法が表示されます

## インストールについて
インストールは必要ありません。  
このアプリは Streamlit で作られており、URLを開くだけで利用できます。
